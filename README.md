# Quizdone z.s.

Repozitář se stanovami. Všechny změny budou zaznamenány sem.

Zápisy k soudu budou zaznamenány jako GIT tagy.

Spolek pro podporu vzdělávání v oblasti hudby, IT a digitálních nástrojů.

## 🚀 Co děláme

    Vývoj otevřených vzdělávacích aplikací.

    Kurzy, workshopy a hackathony.

    Podpora moderních a bezpečných technologií.
